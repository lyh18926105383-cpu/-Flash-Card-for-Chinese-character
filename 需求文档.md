# 识字卡 - 需求文档

## 1. 产品概述

一款帮助儿童或汉语学习者学习汉字的工具应用。通过精美的卡片形式展示汉字，支持发音和图片辅助记忆。

## 2. 功能需求

### 2.1 卡片管理
- **添加卡片**：手动输入汉字，触发自动搜索读音和图片
- **删除卡片**：支持删除已有卡片
- **卡片浏览**：左右滑动或点击切换浏览卡片
- **数据持久化**：使用 LocalStorage 保存卡片数据

### 2.2 卡片内容
| 面 | 内容 |
|---|---|
| 正面 | 汉字（大字居中） |
| 反面 | 音频图标、插图、2-3个组词 |

### 2.3 音频播放
- **发音来源**：使用浏览器 Web Speech API（speechSynthesis）合成中文发音
- **交互方式**：点击音频图标播放当前汉字读音
- **语音参数**：lang="zh-CN", 语速适中

### 2.4 图片搜索
- **图片获取**：使用免费图片 API（如 Unsplash）
- **降级处理**：网络失败时使用预设占位图（纯色背景 + 汉字）

## 3. UI/UX 设计

### 3.1 视觉风格
Apple 简约风格（iOS/macOS Design Language）：
- 大量留白
- 圆润卡片（border-radius: 16-20px）
- 柔和阴影
- SF Pro / 苹方字体
- 低饱和度配色

### 3.2 配色方案
```
背景色:    #F5F5F7 (Apple 灰白)
卡片色:    #FFFFFF
主文字:    #1D1D1F
次要文字:  #86868B
强调色:    #007AFF (Apple Blue)
强调色2:   #34C759 (Apple Green)
```

### 3.4 交互设计
- **卡片翻转**：点击卡片中心区域触发 3D 翻转动画
- **翻页**：底部指示点可点击跳转，卡片区域支持手势滑动
- **添加流程**：点击 "+" → 弹出输入框 → 输入汉字 → 自动获取数据 → 确认添加
- **Haptic 反馈**：按钮点击有轻微震动感（移动端）

## 4. 技术实现

### 4.1 技术栈
- 纯 HTML + CSS + JavaScript（单文件）
- 无框架依赖
- Google Fonts（Noto Sans SC 字体）

### 4.2 数据结构
```javascript
{
  cards: [
    {
      id: "uuid",
      character: "汉",
      imageUrl: "https://...",
      words: ["汉字", "汉语", "好汉"]
    }
  ]
}
```

### 4.3 外部 API
- 图片 API：`https://commons.wikimedia.org` (Wikimedia Commons，支持 CORS)
- 发音：使用浏览器内置 Web Speech API（无需外部 API）

## 5. 页面状态

| 状态 | 描述 |
|---|---|
| 空状态 | 无卡片时显示引导添加提示 |
| 正常浏览 | 显示当前卡片 |
| 添加模式 | 弹出输入界面 |
| 加载中 | 获取图片时显示 loading |

## 7. 验收标准

1. 单 HTML 文件可直接用浏览器打开
2. 卡片可正常添加、翻转、删除
3. 点击音频图标可播放汉字发音
4. 网络正常时可自动获取图片
5. 页面风格符合 Apple 简约设计
6. 移动端可正常使用
