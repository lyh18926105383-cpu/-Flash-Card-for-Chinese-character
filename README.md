# 识字卡

一款帮助儿童或汉语学习者学习汉字的工具应用。

## 使用方法

### 直接打开
双击 `index.html` 文件，用浏览器打开即可使用。

### 添加卡片
1. 点击右上角的 **+** 按钮
2. 输入汉字（一次只能输入一个）
3. 系统自动获取配图
4. 点击 **确认添加**

### 学习模式
- **翻转卡片**：点击卡片区域，查看背面（音频+图片+组词）
- **播放发音**：点击卡片背面的音频图标
- **切换卡片**：点击底部圆点或左右滑动

### 编辑与删除
- 点击底部 **编辑** 按钮修改组词
- 点击底部 **删除** 按钮移除卡片

## 功能特点

- 苹果简约风格界面
- 卡片 3D 翻转动画
- TTS 中文发音（点击音频图标播放）
- 自动获取配图
- 本地存储，关闭浏览器不丢失数据
- 支持移动端使用

## 技术说明

- 纯 HTML + CSS + JavaScript，无任何依赖
- 发音使用浏览器 Web Speech API
- 图片来源 Unsplash
- 数据存储在浏览器 LocalStorage

## 注意事项

- 音频功能需要浏览器支持 Web Speech API（Chrome、Safari、Edge 均支持）
- 图片加载需要网络连接，离线时显示占位图
- 数据存储在本地，清除浏览器缓存会丢失数据
